# Molding C8.5 Big Pellet Supply Issue – EQMLD000147

## Issue
- Machine: Molding EQMLD000147
- Model: C8.5 Big
- Time: 10:30 PM (10/10/2022)
- Machine area: Pellet supply

## Symptom
- Pellet supply issue occurred at the molding machine.
- Machine alarm reported pellet removal / pellet abnormality in pellet supply.

## Root Cause
- Machine alarm: remove pellet which was dumped in pellet supply.
- Pellet insert was not good.
- Because the pellet insert was not good, it could not hold compound correctly.
- Compound dropped during pellet supply into moldchase.

## Failure Mechanism
- Pellet insert function was insufficient to hold / guide compound properly during pellet supply.
- Because the insert did not hold compound correctly, compound dropped at the pellet supply stage.
- This unstable pellet / compound transfer condition created the pellet supply abnormality in the moldchase loading path.

## Actions
- Replaced the pellet insert with a new one.
- Machine could run again from 1:00 AM.

## Current Status
- Pellet insert was changed.
- Machine recovered and resumed running from 1:00 AM.
- Current status: recovered, continue monitoring for repeat pellet-supply abnormality.

## Impact
- Quality: pellet / compound feed instability risk
- Production: machine interruption until pellet insert replacement
- Safety: not stated in the report
- Repeat risk: possible if pellet insert wear / holding condition is not monitored

## Open Risk / Follow-up
- Need to monitor pellet insert condition
- Need to verify pellet supply remains stable in moldchase after replacement
- Need to check if dumped pellet / compound repeat pattern appears again

## Evidence
- Attached report image showing pellet supply area and pellet insert
- Root-cause note states pellet insert was not good and compound dropped during pellet supply
